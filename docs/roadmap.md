# Roadmap propuesto

1. Integrar autenticación y RBAC para perfiles clínicos/investigación.
2. Versionado de modelos (MLflow o DVC).
3. Calibration curve y ajuste de umbral por caso de uso clínico.
4. Módulo de explainability con SHAP.
5. Validación con cohortes reales multi-centro.
6. Auditoría de sesgos por raza/edad/sexo.
7. Exportación PDF de reporte clínico.
8. Dashboard epidemiológico agregado.
